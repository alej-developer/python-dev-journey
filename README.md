# Día 01 — Calculadora de IMC

## ¿Qué hace este programa?
Calcula el Índice de Masa Corporal (IMC) y lo clasifica según los 
rangos de la OMS (bajo peso, normal, sobrepeso, obesidad).

## Conceptos practicados
- Funciones con `def` y type hints
- Entrada de datos con `input()` y conversión con `float()`
- Condicionales `if / elif / else`
- Formateo de texto con f-strings
- Patrón `if __name__ == "__main__"`

## Cómo ejecutarlo
```bash
python imc_calculator.py
```

## Ejemplo de uso
```
Introduce tu peso en kg: 70
Introduce tu altura en metros: 1.75

Tu IMC es: 22.86
Categoría: Peso normal
```
